Mini Program

The following APIs are used to retrieve data collected by app "nRF Interoperability" on WeChat

The information about the interoperability test cases is here.

GET

/api/miniapp/find/report/

Description: Return all reports

/api/miniapp/find/report/brand/

Description: Return all mobile brand

/api/miniapp/find/report/brand/{parameter}

Description: Return reports related to the given brand name.

Example: /api/miniapp/find/report/brand?pm=***

/api/miniapp/find/report/platform/android/

Description: Return all android device report

/api/miniapp/find/report/platform/ios/

Description: Return all iOS device report

/api/miniapp/find/report/sessionid/

Description: Return reports related to the given sessionid.

Example: /api/miniapp/find/report/sessionid?pm=***

nRF91 devkit

APIs in this section are used for nRF91 devkit internet connection test.

GET

/nrf91/test/find/{boardnr}

Description: Return switch status of the given board number

Example: /nrf91/test/find/***

POST

/nrf91/test/push/{boardnr}/{status}/{switchnr}

Description:Push DK(nRF91) swtich status

Type: text/html: charset=utf-8

Parameters: "/boardnr/switchnr/status"

Example: "/1/2/0" means we send the switch number 1 with status is off on board 1
