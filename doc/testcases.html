Inbound Message

Test Number is the linear sequence 1, 2, 3, … for debugging purpose.

Outbound Message

The Thingy BLE interoperability tester has a 5-byte response message formatted as follows:

Test Number

Test Result

Data[0]

Data[1]

Test ID


Test Result has the following notation.

  • 1 passed
  • 0 failed

The tests description are listed in following table.

Test ID

Data[0]

Data[1]

Description

1

Application version

Application version

Returns the application version

2

SoftDevice FWID

SoftDevice FWID

Returns the SoftDevice FWID

3

TX PHY

RX PHY

Request a PHY of 1Mbps or 2Mbps

4

MTU

0

Request a MTU of 247 bytes

5

Data length

0

Request a data length of 251 bytes

6

Connection Interval

Connection Interval

Request a connection interval of 10ms - 30ms

When a data field returned is 16-bit, it is put to Data[0] and Data[1] in Little Endian format.

TX PHY and RX PHY have the following notation.

  • 1 1Mbps
  • 2 2Mbps
