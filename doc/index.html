Welcome to nRF Interoperability project.

Brief

This project started from second-half 2018, aim to perform interoperability test between nRF5x chipset against various mobile devices. The test requires to install a mini-program running on "WeChat" app and a test board with special designed firmware. A report will be sent to a server after test, it contains information about bluetooth feature capabilities, limitation etc.

Privacy

Nordic Semiconductor will collect, in addition to the tests results, the following information: date & time, user country, mobile phone model & OS version, WeChat version - None of the information collected can lead to user identification.
